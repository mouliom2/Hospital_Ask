Architecture technique

Frontend

  • Framework : React 18
  • Langage : TypeScript
  • Style : Tailwind CSS
  • Gestion d'état : Context API et React Hooks
  • Composants : Lucide React (icônes)

Le frontend gère l'interface utilisateur, les interactions, et la communication avec le backend via des appels API.

Backend

  • Environnement : Node.js
  • Framework : Express.js (ou similaire)
  • Base de données : MongoDB ou PostgreSQL
  • API : REST ou GraphQL pour la communication avec le frontend
  • Sécurité : Authentification JWT, chiffrement des données

Le backend gère la logique métier, les données des utilisateurs, et les appels aux modèles d'analyse IA.

Modèles d'analyse IA

  • Frameworks : TensorFlow.js ou Python (TensorFlow, scikit-learn)
  • Types de modèles :
    • Prédictions : Modèles supervisés (régression, classification)
    • Clustering : Modèles non supervisés (K-Means, DBSCAN)
    • Analyse avancée : Réseaux de neurones pour l'aide au diagnostic
  • Communication : Les modèles sont appelés via des microservices ou des API dédiées

Les modèles d'IA analysent les données collectées pour fournir des prédictions, des recommandations, et des insights.
